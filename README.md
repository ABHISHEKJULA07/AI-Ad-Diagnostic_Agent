# 🚀 AI Ad Diagnostic Agent Web Application

An AI-powered web application that helps advertisers diagnose and optimize Meta Ads and Google Ads campaigns using Gemini AI.

Built with Python, Gradio, and Generative AI, this application analyzes advertising issues and provides actionable recommendations for improving campaign performance.

---

# 🌐 Features

✅ Interactive AI-powered web interface  
✅ Meta Ads campaign diagnostics  
✅ Google Ads campaign diagnostics  
✅ AI-generated optimization recommendations  
✅ Real-time campaign issue analysis  
✅ User-friendly Gradio interface  
✅ Fast Gemini AI responses  
✅ Example campaign troubleshooting workflows  

---

# 🧠 What This Web App Can Diagnose

The AI Ad Diagnostic Agent can analyze:

- Ads not spending
- High CPC (Cost Per Click)
- Low CTR (Click Through Rate)
- Ad rejection issues
- Audience targeting problems
- Weak creatives
- Budget allocation problems
- Campaign scaling issues

# ⚙️ Installation Guide

## 1️⃣ Clone Repository

```bash
git clone https://github.com/ABHISHEKJULA07/AI-Ad-Diagnostic_Agent.git
```

## 2️⃣ Navigate to Project Folder

```bash
cd AI-Ad-Diagnostic_Agent
```

## 3️⃣ Create Virtual Environment

### Windows

```bash
python -m venv venv
```

Activate virtual environment:

```bash
venv\Scripts\activate
```

## 4️⃣ Install Dependencies

```bash
pip install -r requirements.txt
```

---

# 🔑 Gemini API Configuration

Create a `.env` file in the project root:

```env
GOOGLE_API_KEY=your_gemini_api_key
```

Get your Gemini API key from:

https://aistudio.google.com/app/apikey

---

# ▶️ Run the Web Application

```bash
python app.py
```

After running, open:

```bash
http://127.0.0.1:7860
```
